+  `git init`：初始化 git 仓库

+ `git clone 代码仓库地址`  ：初始化 git 仓库

+ `git log` ：获取提交历史信息（`commit` 的  SHA-1 校验值（ID）、作者、日期）

+ `git status`：查看工作目录当前状态。
  + 目前 HEAD 在哪个 `branch`
  + 未追踪状态的文件。（untracked）
