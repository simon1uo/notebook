## 概念

弹性布局（Flexible Box，`flex`）可以帮助我们简单、完整、响应式的实现各种页面的布局。

采用 `flex` 布局的元素，称为 `flex` 容器（container）。容器中所有的子元素会自动称为容器的成员，称为 `flex` 项目（item）。

容器中默认存在两条轴，主轴和交叉轴，呈九十度关系。项目默认沿着主轴排列。通过 `flex-direction` 决定主轴的方向。
