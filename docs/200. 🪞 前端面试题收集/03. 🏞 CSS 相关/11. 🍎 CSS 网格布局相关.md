## 概念

Grid 网格布局是一种二维的布局方式，由纵横交错的两组网格形成的框架性布局结构，能够同时处理行与列；

可以将一个页面划分为几个主要的区域，以及定义这些区域的大小、位置、层次等关系。

与 Flex 弹性布局不同，设置为 `display: grid | inline-grid` 的元素为网格布局容器。
