## ES6 `Map` / `Set`

`Map` 字典：是一些元素的集合，每个元素都有一个 `key` ，不同的元素的 `key` 不同。

`Set` 集合：由一堆无序、相关联，并且不重复的内存结构（元素）组成的组合。

两者的区别：

+ 共同点：集合和字典都可以存储不重复的值；
+ 不同点：集合以 `[value, value]` 的形式存储元素；字典以 `[key, value]` 的形式存储。
