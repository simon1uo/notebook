Commitizen 是一个规范 Git Commit 信息的格式化工具，统一项目中的 Commit 信息，便于 Git 版本的管理和日志生成。

### 安装 / 配置

（参考 #Making your repo Commitizen friendly）

在项目中安装 ：

```shell
npm install commitizen -g
```
